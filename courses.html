CMPT 1105

Introduction to Computer Science & Programming

This course was my first ever programming course that (thankfully) was based around Python. I was able to get a hold of the basics of this new syntax, and conceptualize the logic behind the code.

CMPT 1109

Computer Programming 1

The next programming course that I took, which focused heavily on C++. Switching to this strongly typed language was challenging at first, but I eventually found that its uniquely strict language felt surprisingly intuitive after enough practice.

CMPT 1209

Computer Programming 2

1209 was the second half to 1109 in many ways, one of which emphasized the importance of taking an object-oriented approach to any programming task. I find that I do best with structure and routine, so reshaping my 'spaghetti code' mindset was crucial to succeeding in that class.

CMPT 1267

Computer Game Development I

Although a challenging elective, I found that 1267 offered me my first real application of the C++ and Python concepts that I had gained in earlier semesters. It was very satisfying to see my initial vision for my projects take shape in such vibrant ways.

CMPT 2200

Designing with Raspberry Pi

I recall with great clarity how nervious I was when enrolling for this course. Similar to 1267, 2200 was another practical application of the object-oriented programming concepts that I had become familiar with, but this time the projects were based around the Raspberry Pi and its various peripherals.

CMPT 2300

Data Structures and Algorithms

My favourite course by far, and the most challenging I've taken at Douglas College prior to transferring. 2300 really pushed me to scale up my conceptual thinking in order to truly grasp the data structures and algorithms that were being discussed. This course relied heavily on discrete mathematics, which further added to the challenge.
